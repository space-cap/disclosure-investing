# 공시 투자 문서 모음

이 문서들은 한국 주식 공시를 보고 투자 아이디어를 찾기 위한 기본 운영 문서입니다.

목표는 두 가지입니다.

1. 단기 공시 이벤트 매매: 공시 직후 시장이 가격을 다시 매기는 구간을 관찰한다.
2. 중장기 공시 투자: 공시를 통해 좋아지는 회사를 남들보다 일찍 발견한다.

공시 투자는 "공시가 나왔다 = 매수"가 아닙니다. 공시 원문을 보고, 숫자로 검증하고, 주가에 이미 반영되었는지 확인한 뒤, 손실 가능성까지 먼저 정리하는 방식이어야 합니다.

## 중요한 전제

이 문서는 투자 공부와 판단 체계를 만들기 위한 자료입니다. 특정 종목의 매수, 매도, 보유를 권유하지 않습니다. 실제 투자는 본인의 자금 상황, 투자 기간, 손실 감내 범위, 세금, 수수료, 유동성 등을 고려해서 결정해야 합니다.
